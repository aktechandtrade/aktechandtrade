# Image Files Required for the Website

To ensure the website displays correctly, please add the following image files to this directory:

1. **hero-bg.jpg**
   - A high-quality background image for the hero section
   - Recommended size: 1920x1080px or larger
   - Dark or desaturated image works best (the overlay will darken it further)
   - Should represent technology, global business, or trade

2. **about-image.jpg**
   - An image showing your team or office
   - Recommended size: 800x600px
   - Professional business setting
   - Will be displayed in the About section

3. **client1.jpg**
   - Headshot of a person for testimonial
   - Recommended size: 200x200px
   - Square aspect ratio
   - Professional portrait

4. **client2.jpg**
   - Headshot of another person for testimonial
   - Recommended size: 200x200px
   - Square aspect ratio
   - Professional portrait

## Image Sources

If you don't have your own images, you can find suitable free stock photos at:
- [Unsplash](https://unsplash.com/)
- [Pexels](https://www.pexels.com/)
- [Pixabay](https://pixabay.com/)

When using stock photos, be sure to check their license terms and provide attribution if required.

## Image Optimization

For optimal website performance, it's recommended to compress your images before uploading them:
- [TinyPNG](https://tinypng.com/) - For JPG and PNG compression
- [Squoosh](https://squoosh.app/) - For advanced image optimization

## Important Note

The website uses these specific image filenames in the HTML and CSS code. If you use different filenames, you'll need to update the references in:
- `index.html` for the testimonial and about section images
- `css/styles.css` for the hero background image
